⬡ LATTICE SYSTEM REFERENCE
Network Nodes
ALTIUS / ANVIL / ROADRUNNER squads Autonomous Lattice effector nodes — drone units that share a common operating picture, self-coordinate targeting, and report track state continuously to the mesh. Roadrunner deploys 2 precision interceptors per launch; Altius deploys 15 anti-UAS drones; Anvil deploys 8 attrition drones.
HOSTILE TRACK Track object in the Lattice data model — a fused contact with position, velocity, and classification attributes propagated across all nodes.
UNKNOWN / IDENTIFIED IFF pipeline — Lattice classifies contacts from UNKNOWN → SUSPECT → HOSTILE through sensor fusion and rules of engagement logic before authorizing engagement.

PACKET (swarm) Low-cost UAS swarm — small, fast attritable drones designed to saturate defenses. High volume, low signature.
LOITERER Loitering munition — autonomous platform that orbits a target area before terminal engagement. Slow but high-value threat.
Weapons & Effectors
EMP INTERDICTION Electronic warfare payload — area denial through directed energy or RF jamming. Lattice nodes can coordinate EW effects across the battlespace.
CLG GRID Layered area denial system — analogous to Anduril's Lattice-coordinated kinetic/directed energy perimeter. Manages a fixed engagement zone with thermal constraints.
FURY AIR SUPPORT CAS request via Lattice — autonomous air wing tasked through the mesh. Lattice enables operators to request and manage air assets without voice comms to pilots.
C2 & ISR Systems
TTU LINK Tactical mesh uplink — direct telemetry synchronization across all nodes. In Lattice, this represents the data-link layer that enables shared track state and coordinated targeting.
XRST-01 → XRST-04 Extended Range Sentry Towers — fixed 80-foot sensor nodes. Detect, classify, and track threats at ranges up to 7.5 miles autonomously. Feed classified tracks directly into the Lattice mesh. Go offline if breached.
ORS SCAN Overhead persistent ISR — satellite or high-altitude sensor pass. Restores all offline XRST towers and repairs 20% base integrity. Triggers enemy speed increase as a trade-off for the sensor exposure.
SIMULATE JAMMING / ECM Spectrum denial / Electronic Counter Measures — adversary degradation of the data link. Lattice is designed to be resilient to comms-denied environments through autonomous node behavior.
Kill Feed log Real-time track state log — every engagement event, classification change, and system state update propagated to all nodes in the mesh simultaneously.
Logistics
LOITER / RE-ENGAGE cycle Autonomous persistence loop — drone nodes loiter at intercept-ready distance when no target is available, immediately re-engaging when a new contact is authorized. 20% of each squad holds a tight reserve orbit around the asset as a last-ditch intercept layer.
AUTO-TASKING ENGINE Autonomous decision layer — scores all identified threats by proximity, threat level, and time-to-impact. Automatically assigns effectors and deploys squadrons when operator enables AUTO mode. Player can override at any time.
Anduril's Lattice is a software-defined command and control platform that fuses sensor data from heterogeneous assets into a single operational picture, enabling autonomous and semi-autonomous engagement decisions. This simulation models the coordination layer, not individual hardware.
